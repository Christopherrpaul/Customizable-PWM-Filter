1. Reference the article at https://www.edn.com/custom-design-pwm-filters-easily/ .

2. Download the project from GitHub:
   - Click "Code" → "Download ZIP"

3. Extract the ZIP file:
   - Right-click the ZIP → "Extract All"

4. Move the entire extracted folder to a permanent location which is NOT supported by OneDrive or similar applications. 
OneDrive forces references to files at https: locations. The downloaded spreadsheet application needs to create text files and can only access them in folders on a local computer.

5. Open the folder and the .xlsm file. If you want to run the LTspice filter simulations (optional) you must state the location on your computer of the LTspice.exe file. For example, C:\Users\chris\AppData\Local\Programs\ADI\LTspice\LTspice.exe. The version that this application has been developed with can be downloaded from  https://ltspice.analog.com/software/LTspice64.exe . Install this version in a directory different from ones you might already have.

6. Refer to the article in section 1 above for user instructions.
